This directory contains the iVelocity.

A light template engine that support iOS and VTL. It will useful to 
generate a html from template file and NSDictionary data in iOS. 
And JSON can easy to become a NSDictionary to use. 

Now, the iOS App is more easy than ever!

See the file INSTALL for generic compilation and installation instructions.
See the file doc/FAQ for frequently asked questions.

Please send bug reports to <evanzhou2000@gmail.com>.  Please include the
version number from `ivelocity.h', and the error test case in each bug report.

If you have questions about using iVelocity and the documentation does
not answer them, please send mail to <evanzhou2000@gmail.com>.
