Basic Installation
==================

This project is only opened by xcode. Find iVelocity.xcodeproj in ./iVelocity folder and double click it to open this project by xcode.
 
Compilers and Options
=====================

First you should generate the iVelocityLib, please choose 'iVelocityLib' tab and compile it. And then choose the 'iVelocity' tab to generate test cases iOS app to test the iVelocity Lib that you just create.

Used In Your Project
====================================

Copy the iVelocityLib and iVelocity.h two files to your project folder. Include it in your project.
